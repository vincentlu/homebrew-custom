# homebrew-custom

Custom Homebrew formulae.

## Formulae

### Emacs

GNU Emacs built with the native macOS Cocoa GUI (`--with-ns`).

```
brew install vincentlu/custom/emacs
```

To update:

```
brew update
brew upgrade vincentlu/custom/emacs
```
